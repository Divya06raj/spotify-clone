# 🎵 Spotify UI Clone (HTML & CSS Only)

Welcome to my static Spotify clone! 🎧 I built this project purely to challenge my frontend styling skills and create a pixel-perfect layout without using any JavaScript or external frameworks.

### 🛠️ Tech Stack
* **HTML5:** For the page structure.
* **CSS3:** For the dark theme, hover animations, and complex layouts using Flexbox & Grid.

### ✨ What's Inside?
* A visually accurate replica of the Spotify Web Player UI.
* Clean and organized CSS without relying on Bootstrap or Tailwind.
* Smooth hover effects on cards, buttons, and sidebars to make it feel real!

*(Note: Since this is purely HTML/CSS, it's a static UI and does not play music!)*
